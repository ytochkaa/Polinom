МНОГОЧЛЕН
Этот репозиторий содержит реализацию основных методов теории чисел и работы с многочленами над конечными полями, применяемых в криптографии. Основная цель — изучение и практическая реализация операций с многочленами, проверка их неприводимости и выполнение арифметики по модулю простого числа.
